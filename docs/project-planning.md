# 📋 Planejamento do Projeto

## 1. Informações Gerais

### Nome

TaskFlow

### Categoria

Sistema Web de Gerenciamento de Tarefas

### Versão

1.0.0

### Status

Em desenvolvimento

---

## 2. Contexto

A empresa fictícia TechFlow Solutions foi contratada para desenvolver um sistema de gerenciamento de tarefas para uma startup de logística. O objetivo é permitir o acompanhamento das atividades em tempo real, facilitar a organização das tarefas e aplicar conceitos de Engenharia de Software durante todo o ciclo de desenvolvimento.

---

## 3. Problema

Pequenas equipes frequentemente utilizam planilhas ou anotações para controlar tarefas, dificultando o acompanhamento do progresso e a organização do fluxo de trabalho.

---

## 4. Solução

Desenvolver uma aplicação web simples que permita cadastrar, editar, visualizar e concluir tarefas de forma organizada, utilizando práticas modernas de desenvolvimento de software.

---

## 5. Missão

Disponibilizar uma aplicação simples e organizada para gerenciamento de tarefas, aplicando boas práticas de Engenharia de Software.

---

## 6. Visão

Demonstrar, por meio de um projeto prático, a aplicação das principais etapas do desenvolvimento de software utilizando metodologias ágeis.

---

## 7. Valores

- Organização
- Simplicidade
- Qualidade
- Colaboração
- Evolução Contínua
